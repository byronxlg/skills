# Project

- **Name:** agentlog
- **Repo:** agentlog
- **Description:** A local-first, framework-agnostic decision log daemon for agentic workflows. Captures agent decisions and reasoning - the missing layer between git history and LLM traces.
- **Tech stack:** Go (daemon), SQLite (index), JSONL (log format), Python SDK, TypeScript SDK, TOML (config)

## Platform

| Concept | Tool | Details |
|---------|------|---------|
| Repository | GitHub | agentlog |
| Project board | GitHub Projects | Column per workflow state |
| Issues | GitHub Issues | Sub-issues for epics |
| Code reviews | GitHub Pull Requests | Squash merge to main |
| Discussion channels | GitHub Discussions | One category per channel (Initiatives, Engineering, Human, Public, Status Updates) |
| CI | GitHub Actions | Required checks before merge |
